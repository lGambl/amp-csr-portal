# AMP CSR Portal

An internal Customer Service Representative dashboard for managing customer accounts. Built with React and Material UI, the app provides a single-page interface for viewing customers, inspecting account details, purchase history, and vehicle information.

## Tech Stack

- **React 19** with Vite
- **Material UI 9** (MUI) for UI components
- **CSS Modules** for component styling
- **Mock data** — no backend required

## Getting Started

```bash
npm install
npm run dev
```

## Available Scripts

| Command | Description |
|---------|-------------|
| `npm run dev` | Start the development server |
| `npm run build` | Build for production |
| `npm run preview` | Preview the production build |
| `npm run lint` | Run ESLint |
